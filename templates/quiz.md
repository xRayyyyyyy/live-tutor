# 测验模板

## 一、入学摸底测评

### 快速摸底模式（v1.2 新增）

```markdown
# 入学摸底测评（快速模式） — {科目名称}

> 共 5 题（概念理解题），约 5 分钟完成。
> 本测评基于章节标题和核心概念出题，用于快速判断起始等级。

**1.** [第X章相关] {概念理解题}
   A) ...  B) ...  C) ...  D) ...

**2.** [第X章相关] {概念理解题}
   A) ...  B) ...  C) ...  D) ...

**3.** [第X章相关] 用一句话解释：{核心概念}？

**4.** [第X章相关] {概念理解题}
   A) ...  B) ...  C) ...  D) ...

**5.** [第X章相关] {核心公式或定理} 的含义是什么？
```

```markdown

> 共 {N} 题（{M} 选择 + {K} 简答），满分 100 分。请逐题作答。

## 选择题（每题 {score} 分，共 {total} 分）

**1.** [第X章] 题目描述...
   A) ...  B) ...  C) ...  D) ...

**2.** [第X章] 题目描述...
   A) ...  B) ...  C) ...  D) ...

...

## 简答题（每题 {score} 分，共 {total} 分）

**{N}.** [第X章] 题目描述...
```

### 摸底测评输出

```
📊 摸底测评结果

  选择题：{score}/{max} 分
  简答题：{score}/{max} 分
  ─────────────
  总分：{total}/100 分
  等级：L{N} {模式} ({分数档})

  ✅ 已掌握领域：{topics}
  ⚠️ 需加强领域：{topics}

  将以"{模式}"开始学习，知识点覆盖约 {%}。
```

### WebSearch 试卷搜索规则（v1.2 新增）

搜索试卷时遵循以下过滤规则：

1. **域名优先**：优先 `.edu.cn`、`.edu` 等教育机构域名
2. **格式优先**：优先 PDF 格式的正式试卷
3. **时效性**：优先近 5 年内的试卷
4. **关键词匹配**：搜索词必须包含科目全称
5. **质量校验**：搜索到的题目必须与教材知识点比对后才可纳入
   - 比对方法：检查题目涉及的知识点是否在本教材的知识点列表中
   - 不匹配的题目标注「超纲」并排除
6. **回退策略**：如果搜索结果不足 3 条有效题目，回退到纯自生成题目

搜索查询模板：
- `"{科目全称} 期末考试试卷 site:edu.cn"`
- `"{科目全称} {大学名} 期末试题"`
- `"{科目全称} 考试真题 filetype:pdf"`
