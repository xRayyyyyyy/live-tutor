# 考前备忘录模板（v1.2 新增，v1.6.1 HTML 化）

## HTML 模式（v1.6.1 新增）

当 `render_mode = "html"` 时，备忘录输出为独立 HTML5 文档：
- MathJax CDN + 暗色主题 CSS + `open` 自动打开
- 头部：金黄色调 header（`linear-gradient(135deg,#92400e,#0f172a)`）
- 各段用 `<div class="card">` 分隔：必背公式一览 / 高频错题 TOP N / 易混淆概念对比 / 学习轨迹 / 最后提醒
- 文件命名：`考前备忘录.html`

## Markdown 模式

