# Live-Tutor v1.7.0 — 双模式工作流对比报告

---

## 一、两种模式总览

| | 🅰️ 备考模式 | 🅱️ 学习模式 |
|---|---|---|
| **用户** | 有教材、要考试的学生 | 想学任何知识的普通人 |
| **输入** | PDF / PPT / Markdown 教材文件 | 任意关键词（如"傅里叶变换"） |
| **验证方式** | 章节测验 + 错题集 + 终考 | 费曼循环 + 边界探测 + 速查表 |
| **节奏** | 按章节逐章推进 | 按主题灵活学习，每 3 个一轮 |
| **最终产出** | 全书考试 + 考前备忘录 | 学习总结 + 速查表合集 |
| **等级系统** | L1-L4（四级，按章独立） | Level 1-5（五级，全局） |

## 二、输入对比

### 🅰️ 备考模式的输入

```
用户操作                          系统行为
─────────                        ─────────
上传 PDF/PPT/Markdown     →     自动检测为备考模式
  或                             提取教材章节结构
输入科目名 + "备考/复习"   →     AI 生成章节目录
                                 确认考试类型（期末/期中/考研...）
```

**输入源路由**：

| 输入类型 | source_type | 提取方式 |
|---------|-------------|---------|
| 教材 PDF | `pdf` | pymupdf + OCR → 章节 + 知识点 |
| 课件 PPT | `ppt` | python-pptx → 章节 + 知识点 |
| Markdown 文件 | `markdown` | 标题层级解析 → 章节 + 知识点 |
| 仅科目名 | `ai_generated` | AI 生成 + 可选 WebSearch 增强 |

### 🅱️ 学习模式的输入

```
用户操作                          系统行为
─────────                        ─────────
输入关键词（如"傅里叶变换"） →    自动检测为学习模式
                                 确认背景和学习目的
                                 AI 分解为有序主题列表
```

**输入源路由**：

| 输入类型 | source_type | 提取方式 |
|---------|-------------|---------|
| 关键词/主题 | `keyword` | AI 主题分解 + WebSearch 验证 |

### 每个 Session/章节的输出对比

| | 🅰️ 备考模式（每章） | 🅱️ 学习模式（每主题） |
|---|---|---|
| **讲解** | 六段式学习手册（目标→前提→提纲→笔记→重点→自检→词汇表） | 主题讲解（目标→逐知识点展开→实践练习） |
| **验证** | 章节小测（MCQ + 简答），双 Tab 锁定 | 费曼挑战（可选）+ 自评 |
| **输出文件** | `L{N}_{第X章}_{模式}.html`（手册+小测合一） | 讲解 HTML + `速查表_{主题}.html` |
| **错题处理** | 生成错题集，注入下次重学 | 无错题集，漏洞写入 knowledge_tracking |
| **SM-2** | 测验后 init ⭐⭐⭐ 知识点 | 费曼评分/自评后 init 知识点 |

## 四、验证机制对比

### 🅰️ 备考模式：测验驱动

```
章节小测（每章一次）
  ├─ MCQ：JS 自动评分
  ├─ 简答：LLM 四维评判（准确性40% + 逻辑30% + 公式20% + 表达10%）
  ├─ 总分 → 映射等级（L1-L4）
  ├─ 错题 → 错题集 + 注入重学
  └─ 终考：全书综合 + 20-30% 错题改编
```

### 🅱️ 学习模式：边界探测驱动

```
边界探测（每 3 个 session 一次）
  ├─ 7 题渐进简答（全部主观题，禁止 MCQ）
  ├─ 动态难度：答好加难，答差降难
  ├─ 每题打分（0-10）+ 指出正确/漏洞
  ├─ 答错 → 费曼循环补讲（最多 3 轮）
  ├─ 总分 → 级别调整（升级/保持/降级）
  └─ 薄弱内容注入下次 session
```

**7 题难度分布**：

```
Q1 [当前级-1]  →  Q2-Q3 [当前级]  →  Q4 [试探]  →  Q5 [探测]  →  Q6-Q7 [动态]
   热身              确认基础            试探上限      探测边界       精确定位
```

**级别判定**：

| Q1-Q3 得分 | Q4-Q5 得分 | 结果 |
|-----------|-----------|------|
| 均 ≥ 7 | 均 ≥ 6 | **升级 ↑** |
| 均 ≥ 7 | 均 ≤ 4 | **保持 →** |
| 有 ≤ 4 | — | **降级 ↓** |

## 六、共享组件

以下组件在两种模式中完全共用：

| 组件 | 功能 |
|------|------|
| **Persona 系统** | 5 种教学风格（A-E 选择/切换） |
| **IP 角色语音** | 4 种角色语气（懒羊羊/哪吒/小小怪/灰太狼） |
| **渲染模式** | Markdown / HTML（暗色主题 + MathJax + Mermaid） |
| **SM-2 间隔复习** | review_calc.py 驱动的知识点追踪和复习提醒 |
| **费曼循环** | 通俗解释 → 用户复述 → 找漏洞 → 补讲（最多 3 轮） |
| **20 小时计划** | 核心 20% + 10 节课 + 精准资源推荐 |
| **资源策展** | 全局资源（Step 0）+ 精准资源（每 session） |
| **progress.json** | 统一的进度跟踪数据结构 |

## 七、关键差异总结

| 维度 | 🅰️ 备考模式 | 🅱️ 学习模式 |
|------|------------|------------|
| **驱动力** | 考试压力 → 分数导向 | 好奇心/需要 → 理解导向 |
| **内容来源** | 教材（PDF/PPT/MD）或 AI 生成 | 关键词 → AI 主题分解 |
| **组织结构** | 按教材章节 | 按认知递进（学习阶梯） |
| **学习节奏** | 逐章线性推进 | 每 3 个主题一轮循环 |
| **验证方式** | 标准化测验 + 打分 | 渐进探测 + 费曼复述 |
| **错误处理** | 错题集 + 重学 | 费曼补漏 + 注入下次 |
| **压缩工具** | 考前备忘录 | 速查表 + 合并速查表 |
| **终点** | 终考（全书综合检测） | 完成报告（知识地图 + 建议） |
| **等级调整** | 测验分数 → L1-L4 | 边界探测 → Level 1-5 |
